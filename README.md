# Basit Ağ Tarama Aracı

Bu proje Python kullanılarak geliştirilmiş basit bir ağ tarama aracıdır.

## Özellikler

* Belirtilen yerel ağdaki IP adreslerini tarar.
* Ping yanıtı veren cihazları listeler.
* Başlangıç seviyesinde ağ programlama öğrenmek için uygundur.

## Kullanım

Programı çalıştırın ve ağ adresinizi girin.

Örnek:

192.168.1

Program aktif cihazları listeleyecektir.

## Teknolojiler

* Python 3
* subprocess modülü

## Uyarı

Bu araç eğitim amaçlıdır ve yalnızca izinli ağlarda kullanılmalıdır.
